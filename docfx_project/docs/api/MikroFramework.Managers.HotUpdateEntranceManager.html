Class HotUpdateEntranceManager

Namespace
MikroFramework.Managers
Assembly
MikroFramework.dll

A convenient Entrance that integrates the complete process of hot-update (initialize Hot-update manager- check new version-download updated files-validate completeness-delete redundant files). The complete cycle will automatically start at the beginning of the released mode, with callback functions for each phrase

public abstract class HotUpdateEntranceManager : EntranceManager, IManager
